Detailed Description

Multiplying unsigned channel values of the same type. Performs scaled multiplication result = a * b / max_value.

Example:

bits8 x=128;
bits8 y=128;
bits8 mul = channel_multiply(x,y);
assert(mul == 64); // 64 = 128 * 128 / 255
